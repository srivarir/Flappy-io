# Flappy Bird Clone (Unity)

This is a simple Flappy Bird-style game I built using Unity while learning game development.

The goal was to understand how basic game mechanics work — like player movement, collision, scoring, and UI — and turn it into a complete playable game.

## 🎮 Features

- Bird movement with flap mechanics  
- Smooth tilt rotation based on velocity  
- Score and high score system  
- Main menu and restart functionality  
- Animated bird (flap system)  
- Background clouds for visual depth  
- Camera shake effect on game over  

## 🛠️ Built With

- Unity (2D)
- C#

## 🚀 How to Run

1. Clone this repository  
2. Open it using Unity Hub  
3. Open `SampleScene`  
4. Press Play  

## 📌 Notes

This project was built as a learning exercise, but I focused on making it feel polished and complete rather than just functional.
